Contributing to BrickedUp Save Studio

Thank you for your interest in contributing. We welcome contributions, but please read and follow these rules before submitting.

How to Contribute

- Fork the repository (if public access is available to you) or coordinate with the maintainers via the BRICKEDNLOADED // BL4 Modding Discord (invite: https://discord.gg/5juY3Kwp).
- Create a branch for your change and open a pull request describing your changes.

Approval Requirement

- All contributions require explicit, written approval by BRICKEDNLOADED before they can be merged into the main repository. Do not assume your PR will be merged without this approval.
- To obtain approval, contact BRICKEDNLOADED via email at BRICKEDNLOADED@GMAIL.COM or by messaging within the BRICKEDNLOADED // BL4 Modding Discord. Approval must be clearly communicated by BRICKEDNLOADED (for example, via a comment from the account or an email reply).
